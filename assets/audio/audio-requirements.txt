AUDIO FORMAT RECOMMENDATIONS:
-----------------------------
- Primary: MP3 (best browser compatibility)
- Secondary: OGG (good compression, open source)
- Fallback: WAV (uncompressed, larger files)

VOLUME GUIDELINES:
------------------
- Keep all audio files normalized to similar volume levels
- UI sounds: lighter, subtle (around -20dB to -15dB)
- Game action sounds: moderate (around -12dB to -8dB)
- Win/celebration sounds: can be louder (around -6dB to -3dB)
- Ambient/background: very quiet (around -25dB to -20dB)

TECHNICAL NOTES:
----------------
- Max file size per audio: 500KB recommended for web performance
- Sample rate: 44.1kHz or 48kHz
- Bit rate: 128kbps to 192kbps for MP3
- Duration: Most effects should be 0.5-3 seconds (except ambient loops)
- Ensure files are web-optimized and compressed

IMPLEMENTATION:
---------------
Audio files will be loaded via HTML5 Audio API and integrated into each game component.
Volume controls and mute options will be available in the UI.
All audio will have fallback silence for users who prefer no sound.
